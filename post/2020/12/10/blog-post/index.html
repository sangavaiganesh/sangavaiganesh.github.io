Reticulate with Python and R

December 10, 2020   

Something that is useful when coding is to be able to use Python in an R interface so that you can use both languages. Just like if you were bilingual, it is useful to be able to switch between the two languages to execute different functions to the best that it could be.

When using reticulate, you can bring information that you used in R to Python and vice versa. Here we are defining "What starts here" as UT in an R code chunk

library(reticulate)
UT<-"What starts here"

We then are saving "changes the world" as UT but this time in a Python code chunk. We can now print both the UT that we defined in R and the UT that we have defined in Python to read "What starts here changes the world" in a Python code chunk.

UT2="changes the world"
print(r.UT,UT2) 
## What starts here changes the world

We can also do the same in combining the UT from the R and the UT from the Python to read the same message in an R code chunk.

cat(c(UT,py$UT2)) 
## What starts here changes the world
